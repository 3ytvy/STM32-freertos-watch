# MPU6050 计步算法项目文档
# 适用芯片：STM32F103
# 传感器：MPU6050
# 功能：三轴加速度计步、动态阈值、动态精度、最活跃轴判断
# 量程配置：ACCEL_CONFIG = 0x04 → ±4g 量程
# 1g 对应数值：8192


==================== 1. 整体功能概述 ====================
本项目实现基于 MPU6050 的加速度计步功能，流程如下：
1. 读取 MPU6050 原始加速度数据
2. 均值滤波 → 平滑数据
3. 动态更新每个周期的最大/最小值
4. 计算动态阈值（上一周期 max + min / 2）
5. 动态精度过滤 → 过滤小抖动，只保留有效动作
6. 判断最活跃轴 → 只在变化最大的轴上计步
7. 阈值穿越判断 → 从上往下穿过阈值 → 计一步


==================== 2. 硬件相关 ====================
MPU6050 加速度量程设置：±4g
寄存器值：ACCEL_CONFIG = 0x04
灵敏度：1g = 8192 原始值
静止 Z 轴理论值：8192 左右


==================== 3. 全局变量说明 ====================
AX、AY、AZ      原始加速度值
fil_ax、ay、az  均值滤波后的值
aax、aay、aaz   经过【动态精度过滤】后的最终阶梯值
threshold_x/y/z 三轴动态阈值
step_count      最终步数


==================== 4. 滤波模块 ====================
功能：4 点均值滤波，减少噪声
filtercnt = 4
输出：fil_ax、fil_ay、fil_az


==================== 5. 动态阈值模块 ====================
每 SAMPLE_SIZE 个采样点更新一次阈值
SAMPLE_SIZE = 100

阈值计算公式：
threshold_x = (上周期max_x + 上周期min_x) / 2

作用：自适应不同姿态、不同动作强度


==================== 6. 动态精度模块 ====================
作用：过滤微小抖动，只有变化够大才更新有效值
相当于“动作门槛”

当前参数（适配 ±4g 量程）：
DYNAMIC_PRECISION_X = 1500
DYNAMIC_PRECISION_Y = 800
DYNAMIC_PRECISION_Z = 800

输出：
slid->old_sample
slid->new_sample
最终打印：aax、aay、aaz


==================== 7. 最活跃轴判断 ====================
作用：只在动作最大的轴上计步，防止误触发

ACTIVE_PRECISION = 3000
含义：动作幅度必须 ≥ 3000 才认为是有效走路动作

判断逻辑：
1. 计算三轴变化幅度（max - min）
2. 选出变化最大的轴
3. 必须 ≥ ACTIVE_PRECISION 才生效
4. 否则不计步


==================== 8. 计步判断逻辑 ====================
最终判步条件（必须同时满足）：
1. 动作幅度够大（过动态精度）
2. 是当前最活跃轴
3. 活跃轴幅度 ≥ ACTIVE_PRECISION
4. 旧值 > 阈值 且 新值 < 阈值（从上往下穿越）

满足 → step_count++


==================== 9. 主函数流程 Step_Counting() ====================
1. 读取 MPU6050 数据
2. 均值滤波
3. 更新当前采样值
4. 更新峰值（max/min）
5. 周期结束 → 更新阈值
6. 动态精度过滤
7. 最活跃轴判断
8. 穿越阈值判步
9. 串口输出调试信息


==================== 10. 调试输出格式 ====================
串口打印顺序：
AX,AY,AZ, fil_ax,fil_ay,fil_az, threshold_x,y,z, aax,aay,aaz, GX,GY,GZ

aax/aay/aaz = 动态精度过滤后最终值


==================== 11. 关键参数说明（适配 ±4g） ====================
SAMPLE_SIZE          100     阈值更新周期
DYNAMIC_PRECISION_X  1500    X轴动态精度
DYNAMIC_PRECISION_Y  800     Y轴动态精度
DYNAMIC_PRECISION_Z  800     Z轴动态精度
ACTIVE_PRECISION     3000    最活跃轴最小幅度

这些参数根据量程变化，±4g 必须用大数值！


==================== 12. 常见问题 ====================
1. 静止乱计数
→ 动态精度 或 ACTIVE_PRECISION 太小

2. 走路不计步
→ 动态精度 或 ACTIVE_PRECISION 太大

3. 数值太大/太小
→ 与 ACCEL_CONFIG 量程直接相关
±2g=16384/g
±4g=8192/g
±8g=4096/g
±16g=2048/g


==================== 13. 逻辑总结（最精简） ====================
滤波 → 动态阈值 → 动态精度防抖 → 最活跃轴筛选 → 下穿阈值判步
三重防抖：动态精度 + 活跃轴幅度 + 阈值穿越
